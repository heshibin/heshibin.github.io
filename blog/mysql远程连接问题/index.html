mysql远程连接问题 is not allowed to connect to this MySQL server

January 29, 2023 · One min read
Ma Chu
天选打工人

message from server: "Host '192.168.7.233' is not allowed to connect to this MySQL server
远程连接数据库时报错

原因​

mysql数据库只允许自身所在的本机器连接，不允许远程连接。

解决​

mysql -u root -p
use mysql;
select host from user where user = 'root';
update user set host = '%' where user = 'root';
flush privileges;
